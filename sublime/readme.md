Markdown语法
===============
1. 标题
---------------
Markdown支持两种标题的语法

1. 类Setext
    * 最高阶标题 --> 在标题下面加上若干`=`即可
    * 第二阶标题 --> 在标题下面加上若干`-`即可
    * ***注意***：任何数量的`=`和任何数量的`-`都可以有同样的效果
2. 类atx
    * 在最侧插入1到6个`#`，对应标题的1到6阶
    * 当然你也可以选择性的闭合类atx样式的标题，`#`决定标题的阶数


2. 段落
--------------
### a. 代码段
#### i. 使用`<code>`定义
* 使用`<code>`定义
* ***注意***：`<code>`和`</code>`要独占一行

#### ii. 使用```
* ```要独占一行，成对出现

#### iii. 使用缩进块
* 每行要缩进4个空格或者使用一个tab
* ***注意***：中间可以有空行

### b. 引用
* 引用`>`
* `> >`表示多层嵌套
